# CKM treasure hunt

## Quick Start

### 1) Create and activate a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2) Install dependencies

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 3) Register the notebook kernel (optional but recommended)

```bash
python -m ipykernel install --user --name ckm-hunt --display-name "Python (ckm-hunt)"
```

### 4) Launch Jupyter and open the notebook

```bash
jupyter notebook CKM_triangle_hunt.ipynb
```

If you use VS Code, open the notebook and select the "Python (ckm-hunt)" kernel.
